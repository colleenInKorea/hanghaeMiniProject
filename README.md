우리가 간단한 프로젝트를 만들어도 같이 만들 사람들을 찾아야하는 어려움이 있습니다.
이를 용이하게 만들 웹프로젝트를 생각하게 되었습니다.
내가 할 수 있는 스택을 보고, 매력적인 프로젝트를 찾아
새로운 사람들과 만남을 시도해 보세요!
